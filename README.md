# ClickAgro 🌿

Plataforma de gestão técnica e microinovações para extensão rural. Desenvolvida para facilitar a comunicação entre técnicos e produtores rurais, permitindo diagnósticos, agendamento de visitas e compartilhamento de materiais técnicos.

## 🚀 Funcionalidades

- **Diagnóstico Inteligente:** Identificação de problemas na propriedade e geração de recomendações.
- **Central de Conteúdos:** Envio rápido de materiais técnicos via WhatsApp.
- **Diário de Bordo:** Registro de visitas mesmo offline (LocalStorage).
- **Banco de Microinovações:** Registro de soluções criativas do campo.
- **Painel do Gestor:** Administração de conteúdos e acompanhamento de metas.

## 🛠️ Tecnologias

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** PHP
- **Banco de Dados:** JSON (Local) / LocalStorage

## ⚙️ Configuração

1. Clone o repositório:
   ```bash
   git clone https://github.com/jayne-medeiros/ClickAgro.git
   ```
2. Certifique-se de que o servidor PHP tem permissão de escrita na pasta raiz para criar o arquivo `conteudos.json` e a pasta `uploads/`.
3. Configure as senhas de acesso nos arquivos `script.js` e `painel.js` (veja comentários nos arquivos).

## 🔒 Segurança

> [!IMPORTANT]
> As senhas de acesso foram removidas por segurança. Para utilizar o sistema, edite as variáveis de usuários autorizados nos arquivos JavaScript.
